Localization HQ Editor

Visual Studio Extension

Localization HQ Editor is WYSIWYG editor extension for Visual Studio (2015 / 2017 / 2019).

Manage all localizable string resources, translate to foreign languages and generate strongly typed C# classes.


Main Features

  • WYSIWYG Editor
  • Dark Mode
  • Plugins
  • Code Generator
  • Visual Studio Template (New Project and New Item)
  • Import Resources (from *.resx files)
