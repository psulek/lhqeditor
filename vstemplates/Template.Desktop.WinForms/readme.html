Localization HQ Editor

Visual Studio Extension / Project Template

Project Template - Windows Forms Application with LHQ

VS2015 VS2017 VS2019

  • This project template is compatible with Visual Studio 2015, 2017 and 2019
  • Project template showcase usage of LHQ Editor in Windows Forms Desktop application

Visual Editor

  • File Strings.lhq (project root) is container for all string resources in application
  • File extension (lhq) is handled by LHQ extension loaded into Visual Studio Editor
  • Visual Studio shows editor pane where you can visually edit all string resources for all languages your application requires

Code Generator

  • File Strings.lhq.tt is responsible for generating NET resource files (*.resx) and strongly typed C# files (*.cs)
  • Each time file Strings.lhq is saved in LHQ editor, code generation is triggered automatically by executing file Strings.lhq.tt (T4 Text Templating)
  • File Strings.lhq.tt can be edited to configure generator settings, like destination folder where to produce generated files, format of output, etc

Integration with Windows Forms Desktop application


Sample project description

File Program.cs :

  • StringsContext.Instance.Culture = new CultureInfo("en");
    Sets current culture to English on StringsContext class.

File Form1.cs :

  • Method Setup()
    Populate combobox control with list of available cultures (stored in Strings.lhq file) accesing
    StringsContext.Instance.AvailableCultures

  • Method BindControls()
    var binding1 = new Binding(nameof(Label.Text), StringsBindingProxy.Instance, 
          StringsKeys.Messages.Welcome);
    lbWelcomeMessage.DataBindings.Add(binding1);

    This will create data binding linked with label control lbWelcomeMessage to resource Messages.Welcome which in turn returns localized text according to current culture.

    DataBinding binding2 shows similar usage but extended with custom formating inside localized text using Format C# event.

  • Method button1_Click()
    var message = Strings.Messages.TitleWithParams(Environment.UserName, 
        "Localization HQ Editor", DateTime.UtcNow);

    This will returns localized text with C# string parameters filled with parameters used to call generated method TitleWithParams.

Other Notes
